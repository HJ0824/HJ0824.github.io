Project Summary

2021-04-22

The world today is not peaceful,it is full of conflict and misunderstanding.As we seek the root of these unpleasure,we find that every nation has it won special culture and the way to think this world.

That’s a good thing,culture needs to be diverse, “To be without some of the things you want is an indispensable part of happiness.” British philosopher Russell once said so.But at the same time, culture also needs communication and integration with the development of Globalization,every different nation are not lonely cultural group scatter in this planet any more,we are the one and we need to understand each other.

With the inspiration of this great aim, I developed this Application whcih can help people from different cultral u need to click the “X”button to login or sign up.

When you skip to the login page,you can input your account and password to login.And if you don’t have an account yet,you can click the “sign up” button below for signing up.

Like the login page,you need yo input your acccunt and password to create your own account.When you sucessed,page will stay 2 seconds and then skip to the main page.

After login,the title bar which is at the top of screen become completed,you can see user name and the specific location where this user is.

According to the city where user is , system will calculate this city’s cultural positions and display them on the screen.Evey position essentially is a button,so user can cilck it for more detailed infomation if he/she is interested.

As wee see, in positon detail page,it shows the detailed cultral information about this place,and user can also convert Chinese information to English.

Click the little planet button on the title bar,user will enter another function system of this App.This system designed as a community for users to ask questions about culture,and if other users are familar with this content,and them may will write the answer.

At the first page，there is a question list, all questions are displayed here and user can search relevant question which he/she is interested.

Click the question title,it will skip to the question detail page.And in this page, we can see detailed content of this question and the answers of the question.
If user want to view one of these answers,he/she can click the answer block and get into the answer detail page where he/she can see the complete asnwer.

User can also write a answer of his own to help asker.
